# n8n Security Automation Lab

## Overview
This project demonstrates basic SOC-style security automation using:
- n8n (Docker)
- AWS EC2
- SSH
- Telegram alerts

## Use Case
Monitor SSH authentication logs on an EC2 instance to detect:
- Failed login attempts
- Brute-force behavior

## Architecture
- n8n runs in Docker on EC2
- SSH used to access system logs
- Alerts sent via Telegram bot

## Skills Demonstrated
- Security log analysis
- Automation & alerting
- Linux & SSH
- Docker
- Cloud (AWS EC2)
